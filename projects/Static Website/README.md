# Project Overview

This project involves designing and building a static website for a restaurant to help solve common operational challenges such as order mix-up,
miscommunication, and inefficient customer interaction.
The goal is was to create a simple, clear and user friendly website whre customers can easlily view the menu, understand available services, and
place or prepare their orders with less confusion. By moving key information online the reataurant can reduce errors and improve overall customer
experience
