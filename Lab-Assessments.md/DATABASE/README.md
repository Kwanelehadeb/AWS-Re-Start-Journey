## Database Table Operations
## Overview

This lab covers basic SQL operations: creating a database, creating and verifying tables, storing data, and deleting tables and databases.

## Objectives

Create and manage SQL databases and tables

Inspect table structure and verify creation

Insert and store data

Delete tables and databases safely

## Tasks

View existing databases (SHOW DATABASES;)

Create a new database world (CREATE DATABASE world;)

Create the country table with proper schema

Verify table creation (SHOW TABLES; and SHOW COLUMNS;)

Correct schema if needed (ALTER TABLE ... RENAME COLUMN)

Insert sample data

Delete tables (DROP TABLE) and database (DROP DATABASE)

## Challenges

Create a table city with Name and Region columns

## Takeaways

Learned to create, inspect, and modify tables

Learned to insert and store data

Learned safe deletion of tables and databases

Understood basic SQL database management workflow
