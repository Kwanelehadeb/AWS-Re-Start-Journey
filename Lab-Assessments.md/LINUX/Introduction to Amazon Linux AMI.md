## Introduction to Amazon Linux AMI
## Overview

This lab introduces Amazon Linux AMI and basic Linux operations in a cloud environment. Students access an Amazon EC2 instance to explore Linux commands, navigate the filesystem, and use the man command to understand system documentation.

## Objectives

Use SSH to access an Amazon Linux AMI within Vocareum labs

Understand the purpose of the man command

Demonstrate the search feature of man pages

Examine man page headers

## Tasks Completed

Accessed the Amazon Linux AMI via SSH.

Explored the Linux filesystem and basic directory operations.

Used the man command to find command syntax and descriptions.

Practiced searching within man pages and examining headers for details.

## Challenges
Everthing was straight forward and clear

## Takeaways

Learned to access and work with a cloud-based Linux instance.

Gained hands-on experience with the man command and Linux documentation.
